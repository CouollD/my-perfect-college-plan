# GDD v4 — 大学生活攻略録
## 「合コンという乱戦」を中心に据えた設計

---

## 1. コアコンセプト（再定義）

**このゲームにしかできないこと：**
合コンという「全員が動く乱戦」の中で、限られたコマンドで介入しながら、友達との関係・出会い・自己認識を積み上げる。

**ペルソナ・ときメモとの差：**
- あっち：「デートして好感度を上げる」がメイン
- このゲーム：「飲み会・合コンという場の中でどう動くか」がメイン

---

## 2. ゲームの全体構造

```
大学1年〜4年（4年間）
│
├─ 【日常フェーズ】← 軽め。繰り返す
│   ├─ 飲み会（友達との関係維持・深める）
│   ├─ バイト・サークル（時間と金を管理）
│   └─ 勉強（テスト期間はデバフ）
│
├─ 【合コンフェーズ】← ここが楽しい。月1〜2回
│   ├─ 編成（誰を呼ぶか）
│   ├─ 乱戦（コマンド選択・友達の暴走・取り合い）
│   └─ 結果（好感度変化・自己認識ヒント）
│
└─ 【エンディング】← 4年間の積み上げで決まる
```

---

## 3. 飲み会 vs 合コン

| | 飲み会 | 合コン |
|---|---|---|
| 頻度 | 高い（日常的） | 低い（特別イベント） |
| 目的 | 友達との関係維持・深める | 新しい出会い |
| メンバー | 固定（いつもの仲間） | 編成（誰を呼ぶか選ぶ） |
| 彼女ができた後 | 普通に参加 | 行きにくくなる |
| ゲームとしての役割 | リソース管理・関係維持 | 乱戦・メインコンテンツ |

### 彼女ができた後の合コン
- 「友達のため」という名目なら参加できる
- ただし彼女との関係値が少し下がる
- 彼女の好感度が低いと「なんで行くの？」イベントが発生

## 4. 日常フェーズ設計

### 月ごとのリソース管理
```
時間（10/月）と金（初期資金）を何に使うか
├─ 飲み会に行く（時間-2、金-3000、友達好感度↑）
├─ バイトを入れる（時間-3、金+バイト代）
├─ 勉強する（時間-2、テスト期間デバフ回避）
├─ サークル活動（時間-2、サークル仲間との関係↑）
└─ 合コンを開催（時間-3、金-5000、メインコンテンツ）
```

### リアルイベント（月ごとに発生）
| 時期 | イベント | 効果 |
|---|---|---|
| 4月 | 入学・新歓 | サークル選択。出会いの基盤 |
| 7月 | 前期テスト | 時間削減。頭低いと追加デバフ |
| 8月 | サークル合宿 | 参加費あり。仲間と一気に深まる |
| 11月 | バイト給料日前 | 金欠。行動選択肢が制限 |
| 12月 | 年末忙しい | バイト仲間が捕まらない |
| 2月 | 後期テスト | 7月と同様 |
| 3月 | 春休み | 時間に余裕。合コンしやすい |

## 5. 合コンフェーズ設計（乱戦システム）

### 基本構造
```
① 編成画面（誰を3人連れて行くか）
② 合コン開始（相手女性3人と対面）
③ ターン制乱戦（5フェーズ × 2〜3ターン）
④ 結果（好感度変化・自己認識ヒント）
```

### 3つの管理軸
| 軸 | 説明 |
|---|---|
| 場の空気ゲージ（0〜100） | 全員に影響。60以上で通常、40以下でコマンド効果半減 |
| 各女性の注目度 | 誰に注目が集まってるか。コマンドの効果に影響 |
| 友達の暴走度（0〜5） | 放置すると上がる。5で独占フラグ成立 |

### ターンの流れ
```
① 自動イベント発生
   「けんたがゆなに話しかけた。ゆなが笑った。」
   → ゆなの注目度↑、けんた暴走度+1

② プレイヤーの選択（AP1消費）
   A. 介入する → コマンド選択
   B. 静観する → 友達の自律行動が強まる
   C. 別の子へ → ターゲット変更

③ 結果テキスト表示
   「レシーブは任せた！→けんたの話をうまく受けて絡んだ。
    ゆなの視線が一瞬こちらに向いた。」
```

### 独占フラグ
- 友達が3ターン以上同じ女性に関わる → 独占フラグ成立
- 独占中はその女性へのアプローチ効果半減
- 「頼む、割り込んでくれ」で解除できるが友達好感度-

## 6. 友達の性格特性

| 特性 | 合コン中の行動 | 放置リスク |
|---|---|---|
| 暴走しやすい | 場を支配し始める | 独占フラグが立ちやすい |
| 惚れっぽい | 特定の女性に即座に入れ込む | 独占フラグが立ちやすい |
| 静観しやすい | ほぼ動かない | 存在感が薄くなる |
| 学歴厨 | 学歴・成績の話を振る | ギャル系に逆効果 |
| 譲れる | プレイヤーにスペースを作る | 使いすぎると申し訳なくなる |

### 「譲る」演出
```
けんたがゆなを独占しそうな瞬間
松本けいた「あ、俺ちょっと席外すわ」
→ プレイヤーにスペースが生まれる
→ けいたへの好感度+20
→ でも使いすぎると次に頼みにくくなる
```

## 7. エンディング設計

4年間の終わりに「何を積み上げたか」で決まる。

| エンディング | 主な条件 | テーマ |
|---|---|---|
| 順風満帆 | 彼女あり＋友達そこそこ＋就活成功 | 全部うまくいった |
| 全てを捧げた恋 | 彼女高好感度＋友達薄い＋就活ギリ | 恋愛に全振り |
| 仲間と青春 | 彼女なし＋友達好感度高い複数人 | 合コンを楽しみ尽くした |
| 親友と二人三脚 | 特定の友達好感度MAX×2人以上 | 一生の友達ができた |
| ソロ充 | 友達少ない＋就活大成功＋金あり | 全部一人でやり切った |
| 孤独な卒業 | 友達好感度低い＋彼女なし＋就活失敗 | 何も積み上げられなかった |
| 留年 | 頭低い＋テスト期間を無視し続けた | やらかした |

### エンディングのポイント
- 「彼女ができたか」だけじゃなく「どんな4年間だったか」で決まる
- 合コンを何回やったか・どう楽しんだかが評価される
- 全部を手に入れるのは難しい。何かを犠牲にしながら選ぶ
