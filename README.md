# 大学生活攻略録

大学入学〜卒業までの4年間、合コンメンバーを編成しながら
「誰かを切って誰かを選ぶ」意思決定ゲーム。

---

## セットアップ

### 必要なもの
- VSCode
- Live Server拡張機能（VSCode拡張機能タブで "Live Server" を検索してインストール）

### 起動方法
1. VSCodeでこのフォルダを開く
2. `index.html` を右クリック →「Open with Live Server」
3. ブラウザが開いてゲームが起動する

---

## フォルダ構成

```
goukon-game/
├── index.html          # タイトル画面
├── css/
│   └── base.css        # 共通デザインシステム
├── data/
│   └── gamedata.js     # キャラクターデータ・GameState
└── screens/
    ├── chara_create.html   # キャラクター設定画面
    └── formation.html      # 合コン編成画面
```
