## 合コン当日シーンの立ち絵表示ルール（v20以降）

```
① 合コン開始時
   女性3人が並んで登場（全員 normal）
   ↓ タップで次へ
   男性2人（友達）が登場
   ↓ タップで次へ
   合コン本編スタート

② ランダムイベント発生時
   そのイベントに関わった人の立ち絵が表示

③ 話しかけるコマンド使用時
   話しかけた相手の立ち絵が表示
   好感度帯に応じて表情が変わる
```

## ゲーム設計サマリー

### コアコンセプト
合コンという「全員が動く乱戦」の中で、限られたコマンドで介入しながら、友達との関係・出会い・自己認識を積み上げる。

### パラメータ（7つ）
コミュ力 / ビジュ / 頭 / 手際 / 空気読み / メンタル / 酒の強さ

### 友達14人の性格特性
暴走しやすい / 惚れっぽい / 静観しやすい / 学歴厨 / 譲れる

### エンディング（7種）
順風満帆 / 全てを捧げた恋 / 仲間と青春 / 親友と二人三脚 / ソロ充 / 孤独な卒業 / 留年
