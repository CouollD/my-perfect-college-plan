# GDD — 大学生活攻略録 v3

> 時間と金が有限な4年間、合コンメンバーを編成しながら「誰かを切って誰かを選ぶ」意思決定ゲーム。だんだん自分のことがわかってくる。

---

## 1. Pitch

**一文**: 幹事として合コンメンバーを編成し、グノーシア的なコマンド選択で当日を乗り切りながら、自分のパラメータの思い込みと実態のズレに気づいていく大学4年間のゲーム。

**ジャンル**: ノベル / リソース管理 / 編成ストラテジー
**テーマ**: 「だんだん自分のことがわかってくる」
**参照**: グノーシア（コマンド感）/ ラブプラス（相手リアクション）/ ウマ娘（育成→当日反映）

---

## 2. Core Loop

```
1. ACTION   → 合コンメンバーを3人編成する
2. FEEDBACK → 当日コマンドを選択して場を動かす
3. REWARD   → 好感度が変化・自分のパラメータが少しずつ開示される
4. REPEAT   → 友達関係を補充しながら次の合コンへ
```

---

## 3. パラメータ設計（7つ）

| パラメータ | 合コンでの効果 | 大学生活での効果 |
|---|---|---|
| コミュ力 | 「ここは俺が行く」の成功率 | サークル・バイトの人間関係 |
| ビジュ | 相手の初期好感度 | 第一印象イベントで有利 |
| 頭 | 「話題を深める」「俺のデータによると…」効果 | テスト期間デバフなし・留年リスクなし |
| 手際 | 会場選択・二次会の流れ | バイト・段取り系イベント |
| 空気読み | 「まだ動くな」の精度・失敗回避 | トラブル回避 |
| メンタル | 失敗後の立て直し | 就活・修羅場イベントで耐性 |
| 酒の強さ | 「何飲む？」イベント・二次会の流れ | 固定値。一緒に飲むまで相手の値は「？」 |

## 4. プレイヤーパラメータ

- **実際値**: ゲーム開始時にランダム内部決定。非公開。
- **自己認識値**: スライダーで設定。
- **乖離の開示**: 合コン・イベントを経て徐々に判明。
  - ほぼ合ってる → うっすら点灯
  - 一部ズレ → 該当パラメータだけ修正表示
  - 全然違う → 全部ひっくり返る演出

## 5. 合コン当日システム

### フェーズ構成
```
① 会場選択（事前・手際依存）
        ↓
② 乾杯・最初の一杯（酒の強さ）
        ↓
③ メインイベント（会話 or ゲーム）
        ↓
④ 二次会どうする？（空気読み・手際）
        ↓
⑤ お開き・連絡先交換（メンタル）
```

### コマンド一覧（12個）

**最初から使える**
| コマンド | 効果 | 依存パラメータ |
|---|---|---|
| ここは俺が行く | 特定の相手に基本アプローチ | コミュ力 |
| まだ動くな | 情報収集。相手の傾向ヒントを得る | 空気読み |
| お前だけを見てる | 1人にフォーカス。他への好感度は上がらない | なし |

**自分パラメータで解放**
| コマンド | 効果 | 解放条件 |
|---|---|---|
| もっと聞かせてよ | 相手の傾向と合うと大ヒット。外すと沈黙 | 頭50以上 |
| 自然に移動成功…？ | 好感度ボーナス。失敗すると引かれる | 空気読み60以上 |
| 俺に質問するな | 好感度が大きく動く。失敗リスクあり | メンタル65以上 |

**合コン回数で解放**
| コマンド | 解放条件 | 効果 |
|---|---|---|
| まだ終わらせない | 3回目以降 | 二次会へ誘う。失敗すると一気に下がる |
| 番号、教えてもらえる？ | 5回目以降 | 好感度の積み上げに依存 |
| 同じ方向っぽくない？ | 最終フェーズ限定 | 最後の印象を決定づける |

**友達好感度で解放（汎用）**
| コマンド | 解放条件 | 効果 |
|---|---|---|
| レシーブは任せた！ | 誰かの好感度40以上 | 振った友達のパラメータが発動 |
| 頼む、割り込んでくれ | 誰かの好感度55以上 | カットイン事前仕込み。空気読み依存で発動 |
| 今日は黒子に徹する | 誰かの好感度50以上 | 全体最適。自分は引く |

## 6. スコア計算

### 全体最適スコア
```
(自分 + 友達3人) の平均パラメータ × 相手との補完係数
```

### 自分最適スコア
```
自分パラメータ / パーティ4人平均 × 100
（自分が相対的に突出しているほど高い）
```

### 補完係数
```
相手女性の上位2パラメータとパーティの同パラメータの一致度
（相手の傾向は事前非公開。ヒントから推測する）
```

## 7. 好感度システム

### 男女ともに好感度を持つ
- 友達（男）→ プレイヤーへの好感度
- 相手女性 → プレイヤーへの好感度

### 好感度の上昇要因
| 行動 | 効果 |
|---|---|
| バイト・サークルで一緒に過ごす | 少しずつ上昇 |
| 合コンで全体最適を取る | 全員が上昇 |
| 合コンで自分最適を取る | 自分狙いの相手は上昇。友達は下がる場合あり |
| 友達固有コマンドを使う | 好感度を消費（一時的に下がる） |
| 疎遠にする | 時間経過で自然減少 |

## 8. 相手女性設計（10人）

### グループ構成
| グループ | 人数 | 出会い経路 |
|---|---|---|
| サークル経由 | 3人 | サークル選択で変わる |
| バイト経由 | 3人 | バイト選択で変わる |
| ゼミ・授業経由 | 2人 | 共通で出会える |
| 合コン新顔 | 2人 | 中盤以降に登場 |

### ヒントシステム
- 相手のパラメータは非公開
- 仲の良い友達（好感度が高い）から噂・傾向としてヒントが出る
- 例：「バイト先の同僚曰く、イケメンの彼氏を切らしたことがない」
- 「俺のデータによると…」コマンドでヒントを1つ追加開示できる
